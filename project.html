Public Transport in London

Introduction

This project analyzes public transport usage in London to uncover trends, assess service efficiency, and highlight accessibility challenges. Through data-driven insights and interactive visualizations, the study provides actionable recommendations to improve transport services.

Data

The data for this project is sourced from Transport for London (TfL) API and the London Datastore. Key datasets include passenger flow, route usage, and accessibility levels.

Visualizations

Interactive visualizations showcasing key trends and insights:

  • Peak Hour Traffic Analysis
  • Route Popularity
  • Accessibility Heatmap
  • Delay and Reliability Trends

Conclusion

The analysis highlights key insights into public transport usage in London, including peak travel hours, underserved areas, and route popularity. Recommendations include adjusting schedules during peak hours and improving accessibility in less-served boroughs.
