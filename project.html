Introduction

What drives audience preferences, and how have movie ratings evolved across genres over time? For making data-driven decisions and meeting the customers expectations, it is essential for the companies in the entertainment industry to understand the movie ratings and audience preferences including their change over time. This analysis, conducted based on the dataset about popular movies from TMDb, delves into key indicators including movie vote averages, genres, release years. This study can be helpful for producers, marketers and movie platforms in shaping their content strategies. Nowadays, the companies including Disney+ and Netflix are rising in popularity. Therefore, analysing genre-specific trends in movie industry can be beneficial to reveal genres performing high consistently, preferences of audience and crucial shifts in ratings for these type of companies in terms of improving the quality of streaming services.

Data

This analysis is implemented based on two main datasets which offer a comprehensive view of audience preferences and trends in the movie industry:

1. TMDb API Dataset: I accessed this dataset via TMDb API. It is a highly reliable source for getting data about movies. Dataset contains important metrics related to key variables such as vote averages, popularity scores, genres, release dates etc. The data covers multiple decades that allows extensive view on audience preferences. With the help of this dataset, I could visualize the trends in movie industry based on key variables. API ensures the real-time updates as well. Therefore, it was beneficial in terms of analysing the current trends as well. I cleaned and prepared this dataset for the analysis utilising Pandas library. Also, for visualizing key trends I employed Altair. Loops were used to handle the large amounts of data.

2. Wikipedia Dataset for Academy Awards and Genres: I used this data by employing the web scraping tool Beautiful Soup to get the data from Wikipedia. The dataset includes information about the Academy Awards for Best International Feature Film and its winners. Then I merged this data with the one from API to understand what is the share of genres among the award winners. In addition, I used GeoJson for mapping geographical patterns.

Challenges

While working on the project, I faced with two main challenges. Firstly, the genres were IDs in API instead of human-readable names which was difficult to interpret. To tackle this, "genres_mapping" dictionary was manually created so that the genre names could be linked to the corresponding IDs. Secondly, the TMDb API had some limitations such as the number of movies being retrieved per request. It required pagination for larger datasets. To handle this issue, I implemented "loop" so that I can fetch multiple pages of data while balancing the usage limits of API and data requirements. This solution helped to get access to meaningful and structured data.

Conclusion

This analysis reveals key insights into movie trends. It shows that popular movies are not always the ones with high rating as there can be other factors that impact it including marketing and star power which needs further research. The genres like Drama, Action and Comedy are dominating in the market whilst Documentary applies for niche audiences. Recommendations: Based on the analysis, it can be recommended that streaming companies should shape their contents by considering underrepresented genres. The analysis can be a foundation for the further research, such as exploring viewer demographics, understanding the impact of streaming platforms on voting rates, understanding what affects the award winning tendency or analysing the role of social media in driving movie popularity.
